# Tailwind CSS - Grid (Project 12)

## Concept First: Why Grid Utilities Matter

Grid is powerful for two-dimensional layouts where both rows and columns are important. Tailwind grid utilities make dashboards, galleries, and card layouts easier to build.

---

## Lesson-wise Notes

### 1. Grid Columns and Rows

Examples:

- `grid`
- `grid-cols-3`
- `grid-rows-4`
- `gap-4`

---

### 2. Column and Row Span

Examples:

- `col-span-2`
- `row-span-2`
- `col-span-3`

Use spans to highlight featured content.

---

### 3. Responsive Grid

Common pattern:

- `grid-cols-1 md:grid-cols-2 lg:grid-cols-3`

This adapts layout cleanly across devices.

---

## Quick Revision Summary

- Use grid when layout needs row and column control.
- `grid-cols-*` defines column count.
- `gap-*` sets spacing between cells.
- `col-span-*` and `row-span-*` make standout blocks.
