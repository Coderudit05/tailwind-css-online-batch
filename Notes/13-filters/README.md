# Tailwind CSS - Filters (Project 13)

## Concept First: Why Filter Utilities Matter

Filters let you visually transform elements without editing source images. They are useful for hover effects, muted states, and creative UI styling.

Common utilities include:

- Blur
- Brightness
- Contrast
- Grayscale
- Invert
- Sepia
- Hue rotation

---

## Lesson-wise Notes

### 1. Blur

Examples:

- `blur-sm`
- `blur`
- `blur-lg`

---

### 2. Brightness and Contrast

Examples:

- `brightness-75`, `brightness-125`
- `contrast-75`, `contrast-150`

---

### 3. Grayscale and Invert

Examples:

- `grayscale`
- `invert`

Useful for disabled/preview states.

---

### 4. Sepia and Hue Rotate

Examples:

- `sepia`
- `hue-rotate-60`
- `hue-rotate-180`

Great for creative color shifting.

---

## Quick Revision Summary

- Filters modify appearance dynamically.
- Combine filters with hover states for interaction.
- Use moderation to keep UI readable and consistent.
