# Tailwind CSS - Typography (Project 05)

## Concept First: Why Typography Utilities Matter

Typography defines hierarchy, readability, and tone. In Tailwind, typography is controlled with small utilities for font family, size, weight, spacing, alignment, decoration, and transform.

Core benefit:

- Fast experimentation without writing custom CSS.
- Predictable scaling from labels to headings.
- Cleaner consistency across your UI.

---

## Lesson-wise Notes

### 1. Font Family

Use:

- `font-sans`
- `font-serif`
- `font-mono`

---

### 2. Font Size Scale

Use size utilities from very small to display text:

- `text-xs` to `text-4xl` (and beyond)

---

### 3. Font Weight and Letter Spacing

Combine weight and tracking for better visual rhythm:

- `font-light`, `font-normal`, `font-semibold`, `font-bold`
- `tracking-tight`, `tracking-normal`, `tracking-wide`, `tracking-wider`

---

### 4. Text Alignment

Use:

- `text-left`
- `text-center`
- `text-right`

---

### 5. Text Decoration

Useful for links, emphasis, and notes:

- `underline`
- `line-through`
- `overline`
- Decoration styles like `decoration-dotted`, `decoration-wavy`
- Offset with `underline-offset-*`

---

### 6. Text Transform

Use:

- `normal-case`
- `uppercase`
- `lowercase`
- `capitalize`

---

## Quick Revision Summary

- Typography utilities provide hierarchy and readability.
- Use size + weight + spacing together, not in isolation.
- Decoration and transform utilities improve labels and links.
- Keep typography consistent for better UX.

---

## Practice Tasks

1. Create a section heading system using `text-xl`, `text-2xl`, and `text-3xl`.
2. Build a label row with uppercase text and wider tracking.
3. Create two links with different decoration styles.
4. Compare readability of `font-sans` and `font-serif` in a paragraph.
