# Tailwind CSS - Backgrounds and Shadows (Project 08)

## Concept First: Why Background and Shadow Utilities Matter

Background and shadow utilities improve visual depth and hierarchy. They help separate sections, highlight call-to-action areas, and make layouts feel polished.

Key idea:

- Background utilities set color, gradient, image behavior, and blend.
- Shadow utilities add elevation and depth cues.

---

## Lesson-wise Notes

### 1. Background Image Behavior

Common classes:

- `bg-cover`
- `bg-center`
- `bg-no-repeat`

---

### 2. Gradients

Examples:

- `bg-gradient-to-r from-cyan-500 to-blue-500`
- `bg-gradient-to-r from-indigo-500 via-purple-500 to-pink-500`

---

### 3. Shadow Scale

Examples:

- `shadow-md`
- `shadow-lg`
- `shadow-xl`
- `shadow-2xl`

---

### 4. Shadow Colors

Examples:

- `shadow-blue-500/50`
- `shadow-red-500/60`

---

### 5. Mix Blend

Example:

- `mix-blend-multiply`

Useful for overlapping shape experiments and creative overlays.

---

## Quick Revision Summary

- Use gradient and image utilities to style section backgrounds.
- Use shadow scale for elevation hierarchy.
- Use shadow color for visual mood and brand accent.
- Use blend modes for creative overlap effects.
